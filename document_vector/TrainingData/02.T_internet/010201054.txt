 # 국내 인터넷 서비스 회사

 한국통신(KORNET)
 코넷(KORNET:Korea Backbone Network for Internet Connectivity)은 한국통신
 에서 제공하는 인터넷서비스로 다른 상업용과는 약간 다른 국가기간통신망으로 시
 작 되었다.
     제공서비스 : 쉘어카운트, SLIP/PPP
     안내       : 관할전화국 영업창구(관할국번 + 0000 문의)
                  영업지원국(318-5921∼7 문의)

 하이텔
 한국 PC통신에서 운영하는 하이텔(HiTel) 사용자에게 제공하는 인터넷서비스로 개
 인 인터넷 사용자에게 직접 연결서비스를 하기도 하고 기존의  하이텔 사용자에게
 서비스를 하기도 한다. 기존의 하이텔 사용자가 인터넷을  이용 하려면 별도로 다
 시 인터넷서비스를 신청 하여야 한다.
 기존 하이텔 사용자에게 제공되는 것은 메뉴방식과 쉘어카운트 방식인데 쉘어카운
 트서비스를 받는 사용자는  SLIP/PPP를 흉내내는  TIA, TSHOST  등의 Pseudo
 SLIP/PPP 프로그램으로 SLIP/PPP와  똑같은 WWW서비스(그래픽베이스)를 받을 
 수 있다.
            제공서비스 : 쉘어카운트, SLIP/PPP
            안내       : 513-2200
                         080-022-01410

 나우누리
 나우콤에서 96년 1월 부터 제공하기  시작한 인터넷 서비스로 메뉴방식과 쉘어카
 운트 방식이 있다.나우누리는 처음에 아이네트와 제휴하여  나우누리 사용자로 하
 여금 인터넷을 이용하도록 하였으나 자체적인 인터넷서비스를 준비하여 96년 1월 
 부터 자체 서비스를 제공하기 시작한 것이다. 접속율이 좋은편이다.

            제공서비스 : 쉘어카운트, SLIP/PPP
            안내 : 590-3800

 
 천리안
 데이컴(DACOM)에서 운영하는 천리안 사용자에게 제공하는 인터넷서비스로  기존
 의 천리안 사용자는 자동으로 인터넷을 사용할 수 있다.  즉 천리안 ID가 바로 인
 터넷 ID가 된다.
 만약 인터넷 ID를 별도로 갖기를 원한다면 일명 '보라'서비스라는 또하나의 인터넷
 서비스에 가입하면 된다.

           제공서비스 : 쉘어카운트, SLIP/PPP
           안내       : 220-5205
