◎ 윈도우즈 3.1에서 접속하기

◇ WWW서비스를 받기위해서는 윈도우즈상에서 인터넷 접속을 도와주는 윈속이라는 프로그램과 네스케이프
와 같은 웹브라우저가 필요해요.
 하이텔이나 나우누리, 천리안에서 먼저 트윈속(twnsck20.exe)이나 다윈속(dws21.exe), 트럼펫 윈속(twsk21f.zip)을 통신으로
부터 받습니다. 여기서 받은 파일은 자동압축해제화일이므로 원하는 디렉토리를 만들어 그것을 거기에 복사한후 실행만 시켜주면
 압축이 풀립니다. 하지만 트럼펫 윈속은 zip화일이라는 압축이므로 pkunzip twsk21f.zip 으로 실행시켜서 압축을 풉니다.

 자 이제 위의 환경이 만들어 졌으면 윈도우즈를 실행시켜보세요. [기본 프로그램] → [파일 관리자] → [파일] → [검색]을 
선택합니다. 그후 검색대상은 winsock.dll, 디렉토리 검색은 C:\WINDOWS 로 하고 하위디렉토리 검색에 클릭해서 마지막으로 확인
을 누르세요. 검색결과에 winsock.dll이 나타나면 [파일] → [삭제]를 눌러서 삭제합니다. 그리고 [파일] → [복사]을 선택하여 
원본에 c:\internet\twsk21f\winsock.dll , 대상에 c:\windows를 쓰고 확인을 눌러 복사합니다.
 통신을 하기위해서는 윈도우즈가 여러분의 모뎀을 인식해야 합니다. 그러기 위해서는 [기본 프로그램] → [제어판] → [포트
]를 선택합니다. 이야기를 실행시켜서 1을 눌러 모뎀이 COM port 몇번에 있는지 확인해보는게 편리할거예요. 자 
확인했으면 여기서 그와 같은 번호를 가진
아이콘에 마우스로 클릭한 후에 설정[S]를 눌러보세요. 전송속도는 19200, 데이터비트는 8, 패리티는 없음, 정지비트는 1, 흐름
조절은 하드웨어로 맞추어서 확인을 눌러 끝냅니다. 포트 설정이 끝났으면 windows를 재부팅합니다. 그런후 [프로그램 관리자] → 
[보조 프로그램] → [터미널]을 선택한다. [설정] → [통신]에서 전과 같이 전송속도는 19200, 데이터비트는 8, 패리티는 없음, 
흐름조절은 하드웨어, 연결은 com4:, 정지비트는 1로 하고 확인을 누르고 터미널창에서 커서가 깜빡일 때 atz를 입력해서 ok가 
나오면 정확히 설정된겁니다.
 이제 준비작업은 끝났습니다. 자 이제 아까 압축풀었던 프로그램을 윈도우즈에 등록하기 위해서 [프로그램 관리자] → [파일
] → [등록]에서 프로그램 그룹 확인을 누르고, 그룹이름에 internet을 넣고 확인을 누르세요. 그러면 프로그램 그룹 창을 열고 
[프로그램 관리자] → [파일] → [등록]을 선택하여 프로그램을 선택한 후 [확인]을 누른다. 그후 경로명에 트럼펫 윈속 프로그
램이 있는 디렉토리명과 실행 파일명(c:\internet\twsk21f\tcpman.exe)을 쓰고 [확인]을 누른다. 자 이제 프로그램이 등록되었습
니다.

 이제 넷스케이프 설치를 해야합니다. 넷스케이프도 트럼펫 윈속을 받을 때처럼 파일을 받고 internet디렉토리 밑에 temp를 
만들어 거기에 카피한후 압축을 풉니다. 그리고 윈도우즈를 실행시킨후 [프로그램 관리자] → [파일] → [실행]에서 경로명에 c:
\internet\temp\setup.exe를 넣고 확인을 누르면 Netscape Setup창이 나오면 [Continue]를 누른다. 다시한번 [Continue]를 누르
면 넷스케이프가 설치될겁니다. README를 읽겠냐고 하면 No라고 대답하면, 인스톨은 끝나게 되요. 설치가 완료되면 넷스케이프 
실행 아이콘이 internt 프로그랩 그룹에 등록되지요. 
 자 이제 인터넷을 접속해 볼까요? 트럼펫 윈속을 실행합니다. [Dialler] → [Manual Login]을 선택하면 화면 하단에 'SLIP D
ISABLED'라고 표시됩니다. 이때 다음 줄에 접속 번호를 입력한다. atdt 766-9696 이렇게 걸면 한글이 깨져서 나옵니다. 그러나 
걱정 마시고 전에 하던데로 아이디와 비밀번호를 넣으면 이상한 글자들이 올라갈 겁니다. go internet을 하고 22을 
선택하고 하이텔에서 부여받은 인터넷 아이디와 비밀번호를 차례대로 넣으면 유닉스 프롬프트가 나왔을 때 'tia'를 입력하고 Esc
키를 누르면 'SLIP ENABLED'라는 메시지가 표시되는데, 이로서 트럼펫 윈속을 통한 인터넷 접속이 완료된 거예요.(나우누리는 go
 internet하고 1을 선택하면 자동으로 'SLIP ENABLED'가 나옵니다.) 그럼 이제 우리는 넷스케이프 실행 아이콘을 두 번 클릭하여
 넷스케이프를 실행시키면 이제 당신은 인터넷에 들어온 겁니다.
