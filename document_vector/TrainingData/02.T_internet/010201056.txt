 홈페이지 제작을 위한 HTML 다루기 
 HTML 문서에 대해서 

    " HyperText Markup Language " 라는 말의 첫 글자를 따서 부르는 축약형 인터넷 
    고유 명사이다. 즉, 하이퍼텍스트를 만들기 위한 문서의 구조를 기술하는 언어이
    다. 

    HyperText는 Windows에서의 도움말 형식의 문서를 말하는 것이다. Windows의 도움
    말을 보면 색깔이 다른 단어를 클릭하면 그 단어에 대한 설명을 보여준다. 이러한
    방법으로 문서들이 동적으로 연결되어 있는 즉, Link되어 있는 문서를 말한다.
    실제로는 HyperMedia라고 하는 것이 더 정확한 표현일 것이다. HTML은 문서뿐만
    아니라 그래픽, 사운드까지 연결하여 사용된다. 

    HTML은 텍스트로 작성되어 있으므로 그 문서의 크기가 작으며, 인터넷 상에서 
    빠르게 전송될 수 있다. HTML 문서는 사용하는 컴퓨터의 기종에 독립적이다. 
    HTML 문서는 어떠한 컴퓨터에서도 보여질 수 있다. HTML 문서를 보기 위해서는
    HTML을 번역하는 browser만 있으면 된다.  

       HTML은 Web browser에게 text를 어떠한 형태로 보여줄지를 알리기 위하여
       Markup Tag를 사용한다. HTML Tag들은 반드시 "" (왼쪽 angle bracket)으
       로 구성되며, Tag의 이름이 뒤에 오고 "" (오른쪽 angle bracket)으로 닫
       는다. 
       끝막음 Tag는 시작 Tag와 bracket 안에 "/"(슬래쉬)가 있다는 것을
       제외하면 비슷하게 보인다. 약간의 예외는 있지만 HTML 태그들은 "Start Tag"        와 "End Tag"의 쌍을 이룬다. 그 형식은 다음과 같다고 정의할 수 있다. 

       Start Tag ... Document String ... /End Tag 
 HTML 문서들은 대문자와 소문자를 구별하지 않는다.  
 문서내의 공백은 무시된다. 

       HTML 문서는 검색기에 의해서 번역될 때 스페이스 문자, 탭 문자, 엔터 문자
       모두를 무시한다. 
 HTML 문서는 아스키(ASCII)라고 알려진 일반 Text 양식의 문서이며 UNIX 기계상의
          Emacs 나 vi 와 같은 Text 편집기 혹은 PC 상에서 수행되는 모든 종류의 Text 
    에디터를 사용하여 작성할 수 있다.
